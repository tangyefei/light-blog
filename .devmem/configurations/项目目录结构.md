# 项目目录结构

## 包结构约定（com.example.demo）

```
com.example.demo
├── common/                  # 通用类
│   ├── Result.java          # 统一响应封装 Result<T>（code/message/data）
│   └── GlobalExceptionHandler.java  # 全局异常处理（@RestControllerAdvice）
├── config/                  # 配置类
│   ├── Knife4jConfig.java   # Knife4j 配置（空类，通过 properties 配置）
│   └── SecurityConfig.java  # 安全配置（BCryptPasswordEncoder Bean）
├── controller/              # 控制器层
│   └── UserController.java
├── dto/                     # 数据传输对象（请求/响应）
│   ├── RegisterRequest.java
│   ├── LoginRequest.java
│   └── LoginResponse.java
├── entity/                  # 数据库实体（@TableName）
│   └── User.java
├── mapper/                  # MyBatis-Plus Mapper（@Mapper + BaseMapper）
│   └── UserMapper.java
├── service/                 # Service 接口
│   ├── UserService.java
│   └── impl/
│       └── UserServiceImpl.java
└── DemoApplication.java     # 主启动类（@MapperScan("com.example.demo.mapper")）
```

## 资源目录

```
src/main/resources/
├── application.properties   # 主配置文件
├── sql/
│   └── init.sql             # 数据库初始化脚本
├── mapper/                  # MyBatis XML（暂无，使用 MyBatis-Plus 注解方式）
├── static/
└── templates/
```

## 统一响应格式

```java
// Result<T> 静态工厂方法
Result.success(data)     // 200 + data
Result.success()         // 200 + null
Result.fail(message)     // 400 + message
Result.fail(code, msg)   // 自定义 code + message
```
