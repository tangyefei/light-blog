# 用户登录业务规则

## 流程

1. 用 account 字段同时匹配 username 或 email（LambdaQueryWrapper.or()）
2. 用户不存在 → 抛 IllegalArgumentException("用户不存在")
3. BCrypt 验证密码（passwordEncoder.matches(明文, 密文)）
4. 密码错误 → 抛 IllegalArgumentException("密码错误")
5. 生成 UUID Token（UUID.randomUUID().toString().replace("-", "")）
6. 更新 user 表的 token 字段和 updatedAt
7. 返回 LoginResponse（id/username/email/avatar/token）

## Token 方案

- 类型：UUID（32 位无横杠字符串）
- 存储：user 表 token 字段（有状态，存库）
- 说明：暂时方案，后续可升级为 JWT
