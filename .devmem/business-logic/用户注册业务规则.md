# 用户注册业务规则

## 流程

1. 校验用户名唯一性（查 DB，已存在则抛 IllegalArgumentException）
2. 校验邮箱唯一性（查 DB，已存在则抛 IllegalArgumentException）
3. 密码使用 BCrypt 加密（BCryptPasswordEncoder.encode）
4. 写入 user 表（createdAt / updatedAt 手动设置为 LocalDateTime.now()）

## 参数校验（Spring Validation）

- username：@NotBlank + @Size(min=2, max=20)
- password：@NotBlank + @Size(min=6, max=50)
- email：@NotBlank + @Email

## 异常处理

- 参数校验失败 → MethodArgumentNotValidException → GlobalExceptionHandler → 400 + 字段错误信息
- 业务校验失败 → IllegalArgumentException → GlobalExceptionHandler → 400 + 错误信息

## 关键实现

- Service：com.example.demo.service.impl.UserServiceImpl#register
- BCrypt Bean：com.example.demo.config.SecurityConfig#passwordEncoder（注入方式，非 new）
